# ChatMe 🚀

Real-time chat application built with Socket.io, Redux Toolkit, and React.

## Setup Instructions

### Backend
1. Go to `Backend` folder.
2. Run `npm install`.
3. Run `npm start` (or `npm run dev` if you have nodemon).

### Frontend
1. Go to `Frontend` folder.
2. Run `npm install`.
3. Run `npm run dev`.

## Project Structure (Moodify/Perplexity Style)
- `Backend/`: Node.js + Express + Socket.io
- `Frontend/`: Vite + React + Redux Toolkit
